============
Installation
============

Use Python 3.12 (recommended)
-----------------------------

.. code-block:: console

    python3.12 -m venv .venv
    source .venv/bin/activate
    python -m pip install --upgrade pip

Install from source
-------------------

.. code-block:: console

    pip install -e .

This installs the ``nuxnet-pred`` command for the minimal NuxNet inference scaffold.
