==================
NuxNet Inference
==================

Inference scaffold for 3D nuclei segmentation using a lightweight mlf-core style package layout.

This refactor intentionally keeps only the core prediction CLI:

* Segmentation prediction CLI: ``nuxnet-pred``

The uncertainty and feature-importance CLIs were removed to minimize the codebase.

Quick start
-----------

Install:

.. code-block:: console

    pip install -e .

Run with autogenerated dummy 3D input and random (untrained) model weights:

.. code-block:: console

    nuxnet-pred -o ./predictions/demo

Run with a custom NumPy 3D input volume (``.npy`` with shape ``D,H,W``):

.. code-block:: console

    nuxnet-pred -i ./input_volume.npy -o ./predictions/demo

Notes
-----

* Current scaffold mode is CPU-friendly and intended for functional smoke testing.
* Legacy TIFF input and Zenodo checkpoint download code paths are intentionally preserved as comments in ``nuxnet_package/cli_pred.py`` for future reactivation.
