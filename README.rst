================
nuxnet-inference
================

A minimal inference-first Python package for 3D nuclei segmentation.

This repository is intentionally streamlined for PyPI publication and only ships:

* a lightweight inference CLI: ``nuxnet-pred``
* packaging metadata via ``pyproject.toml``
* one GitHub Actions workflow for publishing to PyPI

Quick start
-----------

Install locally:

.. code-block:: console

    pip install .

Run with autogenerated dummy 3D input and random model weights:

.. code-block:: console

    nuxnet-pred -o ./predictions/demo

Run with a custom NumPy 3D input volume (``.npy`` with shape ``D,H,W``):

.. code-block:: console

    nuxnet-pred -i ./input_volume.npy -o ./predictions/demo
